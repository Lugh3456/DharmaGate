# The Dharma Gate · 法门

A personal Buddhist scripture study portal hosted on GitHub Pages.

## What it is

A hub-and-spoke portal linking to nine mini sites, each dedicated to one of Buddhism's most important scriptures. Spans the Theravada, Mahayana, and Vajrayana traditions. Presented bilingually in English and Chinese.

## The Nine Scriptures

| # | English | Chinese | Tradition |
|---|---------|---------|-----------|
| 1 | Heart Sutra | 般若波罗蜜多心经 | Mahayana |
| 2 | Diamond Sutra | 金刚般若波罗蜜经 | Mahayana |
| 3 | Lotus Sutra | 妙法莲华经 | Mahayana |
| 4 | Avatamsaka Sutra | 大方广佛华严经 | Mahayana |
| 5 | Vimalakirti Sutra | 维摩诘所说经 | Mahayana |
| 6 | Amitabha Sutra | 阿弥陀经 | Mahayana (Pure Land) |
| 7 | Dhammapada | 法句经 | Theravada |
| 8 | Sutta Nipata | 经集 | Theravada |
| 9 | Bardo Thodol | 中阴得度 | Vajrayana |

## Tech stack

- Pure HTML + CSS, no frameworks
- All styles inline in `index.html`
- PWA-ready (manifest.json + service worker)
- Hosted on GitHub Pages

## Local preview

Open `index.html` directly in a browser. No build step required.

## Deployment

Push `index.html`, `manifest.json`, `sw.js`, and `icons/` to the root of the GitHub repository. GitHub Pages serves it automatically.

Planned URL: `https://lugh3456.github.io/DharmaGate/`
